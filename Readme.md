### 安装包
```
npm install @alicloud/pop-core@1.7.12
npm install @koa/cors@3.4.1
npm install @koa/router@12.0.0
npm install ioredis@5.2.3
npm install joi@17.6.0
npm install jsonwebtoken@8.5.1
npm install koa-body@5.0.0
npm install koa@2.13.4
npm install mongoose@6.6.0
npm install nodemon@2.0.19
```

### 启动
```
npm run dev
```
